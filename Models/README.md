# Models

This folder contains the MATLAB/Simulink models used in this project.

- DAB_Converter_D1.slx – SPS design focused on extending the ZVS range.
- DAB_Converter_D2.slx – SPS design focused on maximizing efficiency.
- DAB_Converter_CL_pure_R1.slx – SPS-controlled CC-CV DAB converter with a resistive load.
